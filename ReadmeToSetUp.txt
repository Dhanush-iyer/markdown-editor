# Real-time Markdown Editor with Live Preview

This project is a real-time Markdown editor built with Node.js and React. It converts Markdown text to HTML in real-time and displays the rendered HTML in a live preview pane. The application includes a backend server for processing Markdown to HTML conversion.

## Features

- **Real-time Markdown Editing**: As you type Markdown in the editor, the HTML preview updates in real-time.
- **Syntax Highlighting (Optional)**: Syntax highlighting in the Markdown editor for better readability.
- **Backend Processing**: A Node.js server handles the Markdown to HTML conversion.

## Prerequisites

- **Node.js**: Make sure you have Node.js installed. You can download it from [Node.js](https://nodejs.org/).

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/markdown-editor.git
cd markdown-editor


### For Backend perform these steps
  Step 1:- Navigate to the backend directory:
  Step 2:- cd backend
  Step 3:- npm install 
  Step 4:- npm start
The server will start on http://localhost:5000.

## For frontend perform these steps
	Step 1:- Open a new terminal and navigate to the frontend directory:cd frontend
	Step 2:- Install the required dependencies:npm install
	Step 3:- Start the frontend application:npm start
The application will start on http://localhost:3000.
